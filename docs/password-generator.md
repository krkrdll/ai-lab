# 01

パスワード生成機能ページを作成してください。
パスワードの長さ、強度（どの文字を使用するか）は設定で変えられるようにしてください。

# 02

レイアウトは2カラム構成にしてください。
左カラムに設定、Generateボタンを配置し、右カラムに生成されたパスワードを10件リスト表示してください。

設定値を変更した際には再生成はせず、Generateボタンをクリックした場合のみGenerateするようにしてください。

# 03

パスワードに含む文字を自由に入力できるカスタム設定を追加してください。

Copy to Clipboardを押した際にポップアップを表示するようにしてください。
