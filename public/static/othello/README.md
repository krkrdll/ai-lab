# ブラウザ オセロゲーム

シンプルでモダンなブラウザベースのオセロ（リバーシ）ゲームです。
HTML、Tailwind CSS、バニラJavaScriptで構築されており、インストール不要ですぐに遊べます。

## 機能

- **完全なオセロロジック**: 
  - 8x8グリッド
  - 合法手の判定（石を挟める場所のみ配置可能）
  - 石の反転処理
  - ターン管理（パス機能含む）
  - 勝敗判定
- **使いやすいUI**:
  - 現在の手番とスコアの表示
  - 置ける場所のヒント表示（ON/OFF切り替え可能）
  - 最後の手のハイライト
  - レスポンシブデザイン（PC/スマホ対応）
- **モダンなデザイン**:
  - Tailwind CSSによるクリーンな見た目
  - 石の3D風スタイリング
  - アニメーション効果

## ファイル構成

```
index.html      # ゲームのメイン構造
css/
  └── style.css # カスタムスタイル（盤面、石、アニメーション）
js/
  └── script.js # ゲームロジック
```

## 遊び方

1. `index.html` をブラウザで開きます（またはPublishタブからデプロイされたサイトにアクセスします）。
2. 黒（先攻）からスタートします。
3. 盤面のハイライトされている場所（または石を挟める場所）をクリックして石を置きます。
4. 相手の石を挟むと自分の色に変わります。
5. 置ける場所がない場合は自動的にパスとなります。
6. 両者ともに置ける場所がなくなるか、盤面が埋まるとゲーム終了です。
7. 石の多い方が勝ちです。
